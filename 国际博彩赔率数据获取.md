# 国际博彩赔率数据获取

本项目新增脚本：

```bash
scripts/generate_international_bookmaker_odds.py
```

它使用 The Odds API 官方接口获取多家国际博彩公司赔率，按本项目 `2026世界杯体彩赔率.md` 里的北京时间和中文对阵做匹配，并写入：

```text
2026世界杯国际博彩赔率.md
```

## API 选择

国际博彩公司接口分两类：

1. 直接官方 API：Betfair Exchange API、Pinnacle API 等。通常需要账号、地区许可、白名单或商业权限。
2. 授权聚合 API：The Odds API、Sportradar 等。更适合本项目这种“多家博彩公司统一拉取、自动刷新”的用途。

不要用绕过风控的网页抓取方式拉取博彩公司赔率；这类方式稳定性差，也可能违反网站条款。

## 环境变量

先申请 The Odds API key，然后设置：

```bash
export ODDS_API_KEY="你的 API key"
```

也可以使用：

```bash
export THE_ODDS_API_KEY="你的 API key"
```

## 常用命令

生成北京时间明天比赛的国际博彩公司赔率：

```bash
python3 scripts/generate_international_bookmaker_odds.py --tomorrow
```

生成指定北京时间日期：

```bash
python3 scripts/generate_international_bookmaker_odds.py --date 2026-06-27
```

只看指定 bookmaker key。key 名称以 The Odds API 当前支持列表为准：

```bash
python3 scripts/generate_international_bookmaker_odds.py --tomorrow --bookmakers pinnacle
```

自动刷新，每 5 分钟写回一次：

```bash
python3 scripts/generate_international_bookmaker_odds.py --tomorrow --watch-interval 300
```

默认 sport key 是：

```text
soccer_fifa_world_cup
```

默认 markets 是：

```text
h2h,spreads,totals
```

`--dry-run` 只是不写文件，仍会请求接口；没有 key 时可用 `--input-json` 调试本地解析。

## 输出字段

输出表列：

```text
时间 | 比赛对阵双方 | 博彩公司 | 类型 | 事件 | 赔率 | 更新时间
```

- `胜负` 使用 `胜 / 平 / 负`，相对于本项目中文对阵顺序。
- `让球` 使用 API 返回的 outcome 盘口线，事件列写具体球队。
- `全场大小球` 使用 API 返回的大小球盘口线。
- `赔率` 是十进制赔率。
- `更新时间` 是接口返回的 bookmaker/market `last_update` 转北京时间。
